# Weighted Grid Pathfinding

**Date:** 2026-07-21  
**Difficulty:** Advanced  
**Estimated time:** 80 minutes

## Brief

Implement pathfinding across a grid containing blocked cells and different movement costs.

## Requirements

- The result must contain an ordered path from start to destination.
- Blocked cells must never appear in the final path.
- Movement cost and estimated remaining cost must be tracked separately.
- The system must report when no path exists.
- The algorithm must avoid repeatedly recalculating closed nodes without cause.

## Additional constraint

All externally visible state must be read-only.
